Test Audio Upload

Open the browser console and upload a file. You should see console logs from the audio store.

Visit: http://localhost:3001

Steps to test:

  1. Open browser DevTools (F12)
  2. Go to Console tab
  3. Click "Upload Audio File" button
  4. Select any audio file
  5. Check console for any errors
  6. Check if waveform appears
  7. Check if playback controls appear

Expected behavior:

  • File uploads successfully
  • Waveform visualization appears
  • AudioPlayer controls appear
  • Play button works
